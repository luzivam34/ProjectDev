Python

Conceito

Python é uma linguagem de programação de alto nível, criada por Guido van Rossum e lançada em 1991. É conhecida por sua simplicidade, clareza e legibilidade, sendo uma das linguagens mais populares do mundo.

Características principais de Python:

Fácil de aprender:

A sintaxe lembra o inglês e é limpa.

Multiparadigma:

Suporta programação orientada a objetos, funcional e procedural.

Grande ecossistema:

Milhares de bibliotecas ( como NumPy, Flask, Django, Pandas, etc).

Interpretada:
Você escreve e executa o código sem compilar antes.

Multiplataforma:
funciona no Windows,Linux, Mac, etc.

Usada em várias áreas:

  • Desenvolvimento Web (com Django, Flask)
  • Ciência de Dados e IA (com Pandas, NumPy, TensorFlow)
  • Automação e scripts
  • Jogos e aplicação desktop

Exemplo simples de código de Python:

nome = input(“Qual é o seu nome? ”)

print(f“Olá, {nome}! ”)
